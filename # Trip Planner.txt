# Trip Planner

A modern web application for planning trips, built with **HTML**, **Tailwind CSS**, **JavaScript**, **React**, and **Node.js**. The application enables users to add and manage their trips with details like destination, start and end dates, and additional notes.

## Features

- **Add New Trip**: Input destination, start and end dates, and notes for your trips.
- **View Trips**: Displays a list of all your planned trips.
- **Responsive Design**: Optimized for both desktop and mobile devices using Tailwind CSS.
